CommandWhoAction

Spotify sync

                
Mood colors + beat pulse for your saved Hue lights

              

              

                Concert mode: lights stay off between hits, then flash color → black on each beat.
                With System audio on, every detected audio beat triggers a flash (not just BPM drops).
              

              

                
              

              

                
              

              

                
              

              

                Audio sensitivity:
                
                6
                
                (higher = more hits)
              

              

                Beat timing fine-tune:
                
                0 offset
                
              

              

                Beat phase nudge:
                
                0 ms
                
              

              
              
              

                Calibrate with Sandstorm: play it from the start — wild mode flashes every beat at 136 BPM.
                Fine-tune with ±5 BPM, then click Save calibration for current song.
              

              

            


            

              

                
Govee LAN

                
Local control via GoveeLAN — no cloud API key needed

              

              

                Your phone uses Govee cloud. This dashboard uses local LAN — in Govee Home → each light → Settings → turn on LAN Control.
                If you run GoveeLAN on this PC, close it first (only one app can use UDP port 4002).
